Using postman tool

- Go to teh option [Collections]
- Import the collection named [DevsuAPi.postman_collection.json]
- Go to the option [Environments]
- Import the collection named [QADevsu.postman_environment.json]
- Go back to the collection option
- On the up-right you need to select the QADevsu environment
- right Click on the collection imported and click Run collection
- Before running collection, check the option persist response is marked, it is located on the right side
- You are going to see the test result


Into the folder you are going to see an HTML report when I test on my side. It is located /newman folder.
